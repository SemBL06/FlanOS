# FlanOS

**Project is under construction!**

FlanOS is an operating system build around the programming language Flanlang. It's main focus is to be as lightweight as possible, while maximising all the possible (hacking) features. It was build as a protest against the high prices companies set for hacking software/hardware. 

Check the documentation to see all the features!
